# О проекте

Проект разработан во **vue 3**, представляет реализацию сайта с заметками (возможности сайта описаны в следующей главе).

# Возможности

- *Регистрация пользователей.*

- *Авторизация пользователя (Авторизация сделана при помощи JWT токена).*

- *Создание заметок.*

- *Редактирование заметок.*

- *Фиксирование выполненных заметок.*

- *Удаление, добавление и фильтрация заметок.*

- *На странице реализована пагинация.*

- *Смена языка и темы проекта.*

# Важно: 

**Проект преднозначен для демонстрации кода рабочего проекта, запустить проект без сервера не получится, по запросу могу предоставить код для запуска**

**Посмотреть функционал проекта можно по ссылке [Github](https://github.com/PavelShlyoma/pdProjectTodosMock.git), проекта с имитацией сервера**
